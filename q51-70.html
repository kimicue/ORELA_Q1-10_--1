✓ a. 30c+5p≤100b. 30c+5p<100 c. 30c+5p>100d. 30c+5p≥100
예산 부등식: 총 비용 ≤ 예산 30c + 5p ≤ 100 c = 잉크 카트리지 수 p = 종이 묶음 수 "may purchase" = 살 수 있음 (= 이하, ≤) 정확히 $100도 가능 → ≤ (같음 포함)
부등호 선택 기준:
"at most / no more than / 이하" → ≤
"less than / 미만" → <

a. -4000b. -400 c. 0✓ d. There is no limit
4x²/(x+2): 분자 차수: x²(2) 분모 차수: x(1) → 분자 차수 > 분모 차수 → 발산 x→-∞일 때: 분자 4x²→+∞ (짝수 지수→항상 양수) 분모 x→-∞ 4x²/(x+2) → +∞/(-∞) → -∞ → 특정 값에 수렴하지 않음 → no limit
x→-∞ 주의:
x²은 x가 음수여도 양수! ((-10)²=100)
x는 음수 그대로
→ 분자+∞, 분모-∞ → -∞로 발산

결론: 분자 차수 > 분모 차수 → 항상 no limit
1
차수 비교: 분자 x²(2) > 분모 x(1) → 발산
2
x→-∞: -∞로 발산 → no limit
