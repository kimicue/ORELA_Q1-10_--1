Number Sense · Closure Under Subtraction

닫힘(Closure) — 뺄셈: 집합 A가 뺄셈에 닫혀 있다 = A의 임의 두 원소의 차도 A에 속함 Natural : 1,2,3,... → 5-10 = -5 → Natural 아님 ✗ Integers : ...-1,0,1,... → 어떤 두 정수의 차도 정수 ✓ Rationals: p/q 형태 → 차도 p/q 형태 ✓ Reals : 차도 실수 ✓

Number Sense · Reverse Percent (Discount + Tax)
A dress is marked down 45%. The cost, after taxes, is $39.95. If the tax rate is 8.75%, what was the original price of the dress?
a. $45.74 b. $58.61 ✓ c. $66.79 d. $72.31
할인 후 세금까지 적용된 최종 가격: 최종가격 = 할인가 × (1 + 세율) = 원가 × (1-할인율) × (1+세율) 역으로 원가 구하기: 원가 = 최종가격 ÷ [(1-할인율) × (1+세율)] = 39.95 ÷ [0.55 × 1.0875] = 39.95 ÷ 0.598125 ≈ $66.79
이 문제는 두 단계가 결합된 역산 문제:
① 45% 할인 → 원가의 55% (=0.55) 만 냄
② 8.75% 세금 → 할인가의 108.75% (=1.0875) 가 최종가

따라서: 최종가 = 원가 × 0.55 × 1.0875
원가 = 39.95 ÷ (0.55 × 1.0875) = 39.95 ÷ 0.598125 ≈ $66.79
1
할인 후 가격: 원가 × (1-0.45) = 0.55 × 원가
2
세금 적용 후: 0.55 × 원가 × 1.0875 = 39.950.598125 × 원가 = 39.95
3
원가 계산:원가 = 39.95 ÷ 0.598125 ≈ $66.79
정답: c. $66.79

Algebra · Geometric Sequence Formula
Which formula represents the sequence 1, 2, 4, 8, 16, ...?
a. y = 2x b. y = x + 2 ✓ c. y = 2^(x-1) d. y = x²
등비수열 (Geometric Sequence): 각 항 = 앞 항 × 공비(r) 1, 2, 4, 8, 16, ... → 공비 r = 2 (매번 2배) → 첫째항 a₁ = 1 일반항: aₙ = a₁ × r^(n-1) = 1 × 2^(n-1) = 2^(n-1) x=1: 2⁰=1 ✓ / x=2: 2¹=2 ✓ / x=3: 2²=4 ✓
등차 vs 등비 구분:
등차수열: 일정한 값을 더함 → 1,3,5,7... (+2)
등비수열: 일정한 값을 곱함 → 1,2,4,8... (×2)

빠른 확인: x=1 대입해서 1이 나오는지 테스트
a: 2×1=2 ✗ / b: 1+2=3 ✗ / c: 2⁰=1 ✓ / d: 1²=1 ✓
x=2 재검: c: 2¹=2 ✓ / d: 2²=4 ✗ → c가 정답!
1
패턴 확인: 1→2→4→8 → 매번 ×2 → 등비수열, r=2
2
공식: aₙ = 2^(n-1)n=1: 2⁰=1 ✓, n=2: 2¹=2 ✓, n=3: 2²=4 ✓
3
선택지 c: y=2^(x-1) → x=1일 때 2⁰=1 ✓
정답: c. y = 2^(x-1)

Algebra · Arithmetic Sequence Formula
Which formula represents the sequence 8, 13, 18, 23, 28, ...?
✓ a. aₙ = 5n + 3; n ∈ ℕ b. aₙ = n + 5; n ∈ ℕ c. aₙ = n + 8; n ∈ ℕ d. aₙ = 5n + 8; n ∈ ℕ
등차수열 (Arithmetic Sequence): aₙ = a₁ + (n-1)d = dn + (a₁ - d) 8, 13, 18, 23, 28, ... 공차 d = 5 (매번 +5) 첫째항 a₁ = 8 aₙ = 5n + (8-5) = 5n + 3 검증: n=1: 5+3=8 ✓ / n=2: 10+3=13 ✓ / n=3: 15+3=18 ✓
등차수열 공식 빠른 도출법:
① 공차(d) 찾기: 13-8=5
② aₙ = 5n + ? 형태 확정
③ n=1 대입해서 상수 구하기: 5(1)+? = 8 → ? = 3
④ aₙ = 5n+3 완성!

d 선택지 확인: 5(1)+8=13 ≠ 8 → 틀림 ✗
1
공차: 13-8=18-13=5 → d=5
2
공식: aₙ = 5n + c, n=1 대입5(1) + c = 8 → c = 3
3
완성: aₙ = 5n+3. 검증:n=2: 10+3=13 ✓ / n=5: 25+3=28 ✓
정답: a. aₙ = 5n + 3

Algebra · Vertex Form of Parabola
Which graph represents y = (x − 4)² + 3?
a. opens up, vertex at (-4, 3) b. opens up, vertex at (-4, -3) ✓ c. opens up, vertex at (4, 3) d. opens down
포물선 꼭짓점 형식 (Vertex Form): y = a(x − h)² + k 꼭짓점(Vertex) = (h, k) ⚠️ 부호 주의: (x-h)에서 h가 양수면 오른쪽 a > 0: 위로 볼록 (opens upward) ∪ a < 0: 아래로 볼록 (opens downward) ∩ y = (x-4)² + 3 → a=1 > 0: 위로 볼록 ∪ → h=4, k=3: 꼭짓점 = (4, 3)
⚠️ 가장 흔한 실수: 부호 반전!
y = (x - 4)² + 3 → 꼭짓점 x = +4 (빼기니까 양수!)
y = (x + 4)² + 3 → 꼭짓점 x = -4 (더하기니까 음수!)

외우는 법: "(x-h)"에서 h의 부호를 반대로!
k는 그대로: +3 → 꼭짓점 y = 3
1
y = (x-4)² + 3 → a=1, h=4, k=3
2
꼭짓점: (h, k) = (4, 3)x=4일 때 최솟값: y=(4-4)²+3=0+3=3 ✓
3
a=1>0 → 위로 볼록(∪ 모양) 포물선
4
→ 꼭짓점 (4,3)에서 위로 볼록인 그래프 = 선택지 c
