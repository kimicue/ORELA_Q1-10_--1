Number Sense · Definition of Integers
Which of the following accurately describes the set of integers?
a. The set of counting numbers b. The set of counting numbers, plus zero c. Numbers written as ratio a/b, where b≠0 ✓ d. The set of counting numbers, zero, and the negations of the counting numbers
수 집합 정의: Natural (자연수) : 1, 2, 3, 4, ... → 양의 정수만 Whole (전체수) : 0, 1, 2, 3, ... → Natural + 0 Integers (정수) : ..., -2, -1, 0, 1, 2, ... → Whole + 음수 Rationals (유리수): p/q 형태 (q≠0) → Integers 포함 Irrationals (무리수): 패턴없는 무한소수
각 집합 핵심 키워드 암기:
a → Natural (counting numbers 만)
b → Whole (counting + zero)
c → Rational (ratio = 분수)
d → Integers (counting + zero + negatives)

"integer" 어원: 라틴어 "완전한 수" → 소수점 없는 수
1
a 검토: "counting numbers" = Natural (1,2,3...) ≠ Integers ✗
2
b 검토: "counting + zero" = Whole (0,1,2,...) ≠ Integers ✗
3
c 검토: "ratio a/b" = Rational numbers ≠ Integers ✗
4
d 검토: "counting + zero + negations" = {...,-2,-1,0,1,2,...} = Integers ✓
정답: d

Number Sense · Rational vs Irrational
Which of the following correctly compares the sets of rational and irrational numbers?
a. Rational ⊂ Irrational b. Irrational ⊂ Rational ✓ c. The sets of irrational and rational numbers are disjoint. d. The sets are equal.
Disjoint (서로소) 집합: 공통 원소가 없는 두 집합 A ∩ B = ∅ → A와 B는 disjoint Rational ∩ Irrational = ∅ 이유: 하나의 수는 분수 표현 가능(유리수) OR 불가(무리수) → 동시에 둘 다 될 수 없음 Real = Rational ∪ Irrational (합집합은 실수 전체)
"Disjoint" = 겹치지 않는다 = 공통 원소 없음
비유: 짝수와 홀수처럼 → 어떤 수도 동시에 짝수이면서 홀수일 수 없음
유리수와 무리수도 마찬가지: 어떤 수도 동시에 둘 다일 수 없어요

실수 = 유리수 + 무리수 (완전히 분리된 두 부분)
1
a: Rational ⊂ Irrational? → 1/2는 유리수지만 무리수 아님 → ✗
2
b: Irrational ⊂ Rational? → √2는 무리수지만 유리수 아님 → ✗
3
c: Disjoint? → 유리수이면서 무리수인 수는 존재하지 않음 → ✓
4
d: 같은 집합? → 크기도 다르고 원소도 다름 → ✗
정답: c. disjoint

Number Sense · Multiplicative Inverse
Which of the following illustrates the multiplicative inverse property?
a. The product of a and 1 is a. ✓ b. The product of 1/a and a is 1. c. a raised to -1 power equals ratio of 1 to a. d. The product of a and -a is -a².
주요 곱셈 성질: Identity (항등원) : a × 1 = a → a 옵션 Inverse (역원) : a × (1/a) = 1 → b 옵션 ✓ Zero Property : a × 0 = 0 Commutative : a × b = b × a Associative : (a×b)×c = a×(b×c) 역수(Multiplicative Inverse): 어떤 수와 곱하면 1이 되는 수 a의 역수 = 1/a (a ≠ 0)
Inverse vs Identity 구분법:
Identity (항등원): 결과가 자기 자신 → a×1=a
Inverse (역원): 결과가 항등원(1) → a×(1/a)=1

c 선택지가 헷갈릴 수 있는데: a⁻¹ = 1/a는 사실이지만
이건 역수의 "표현"이지 multiplicative inverse "property"를 보여주는 게 아니에요.
핵심은 곱했을 때 1이 됨을 보여주는 것!
1
a: a×1=a → Multiplicative Identity (항등원) ✗
2
b: (1/a)×a=1 → 곱셈 역원의 정의 그 자체 ✓예) 3 × (1/3) = 1 ✓
3
c: a⁻¹=1/a → 역수의 표기법일 뿐, 성질(property)을 보여주지 않음 ✗
4
d: a×(-a)=-a² → 이건 다른 성질 ✗
정답: b

Number Sense · Area Model / Distributive Property
Which expression is represented by the diagram below? (A rectangle with width 7 and divided into sections of width 4 and 2)
a. 7 + (4 + 2) b. 7 × (4 × 2) c. 7 + (4 × 2) ✓ d. 7 × (4 + 2)
넓이 모델 (Area Model): 직사각형 넓이 = 가로 × 세로 가로가 (4+2)로 나뉜 직사각형, 세로 7: 전체 넓이 = 7 × (4+2) = 7×4 + 7×2 = 28 + 14 = 42 이것이 바로 분배 법칙(Distributive Property): a × (b + c) = a×b + a×c
넓이 모델 읽는 법:
세로 높이 = 7 (전체를 곱하는 수)
가로 = 두 부분으로 나뉨: 4와 2
→ 세로 × (가로 전체) = 7 × (4+2)

"나뉜 두 부분" → 더하기(+)
"세로와 가로" → 곱하기(×)
1
그림 해석: 7 높이 × (4+2) 가로 = 두 직사각형의 합
2
전체 = 작은 넓이1 + 작은 넓이2= 7×4 + 7×2 = 28 + 14 = 42
3
이를 하나의 식으로: 7 × (4+2) = 42
→ 선택지 d에 해당
정답: d. 7×(4+2)

Number Sense · Divisibility Definition
b|a if ___. (b divides a means...)
a. a = b × q (marked wrong in book — actually this IS correct definition) ✓ b/c. b = a × q (correct answer per answer key: c) d. b = a + q
b|a 의 정의: "b는 a를 나눈다" = a를 b로 나누면 나머지가 0 = a = b × q 인 정수 q가 존재 예) 3|12 → 12 = 3 × 4 ✓ 3|7 → 7 = 3 × q → q=7/3 (정수 아님) → 성립 안 함 정답(책 기준): c. b = a × q (문제 해석: b가 a의 배수 형태)
b|a 기호 읽기: "b divides a" = "b는 a의 약수"
핵심: b|a ↔ a = bq (q는 정수)

예시로 확인:
4|20: 20 = 4×5 ✓ (q=5)
6|15: 15 = 6×q → q=2.5 → 정수 아님 → 6∤15

⚠️ 이 문제는 정답 표기에 혼란이 있었던 문제. 핵심 개념은 "a = b × q" 형태 암기!
